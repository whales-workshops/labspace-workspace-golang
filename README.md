## Release

- Update the `TAG` variable in `docker-bake.hcl`
- `docker buildx bake --push --file docker-bake.hcl`
- Update the `DOCKER_TAG` variable in `release.env`
- Update the `README.md` with the new version
- Run `release.sh`
